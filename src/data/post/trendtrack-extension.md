## Privacy and Permissions: What the TrendTrack Extension Accesses

Chrome extensions request permissions, and savvy users want to know what data an extension can access. The TrendTrack Extension requests several permissions, and here's what each one actually does:

- **activeTab**: Allows the extension to read the current page content when you actively click the extension icon. This is how it detects Shopify stores and extracts store data.
- **storage**: Saves your preferences and research data locally in your browser.
- **tabs**: Enables the extension to detect which websites you're visiting so it can activate the panel on Shopify stores. The TrendTrack Extension does not transmit your full browsing history to its servers — it only processes data from Shopify store pages where you actively use the extension.
- **sidePanel**: Opens the extension interface as a browser sidebar rather than a popup, giving you more screen space for data.

The extension's content scripts match all URLs because it needs to detect Shopify stores across the entire web. However, data processing only occurs when you're on a Shopify store and actively engage the extension. If privacy is a concern, you can restrict the extension's permissions in Chrome's settings to "on click" mode, meaning it only activates when you click its icon.

## Common Issues and Troubleshooting

### The Extension Doesn't Detect Shopify Stores

If the TrendTrack Extension icon doesn't highlight when you visit a Shopify store, try these steps:

1. **Check that the extension is enabled**: Go to `chrome://extensions/` and verify the toggle is on
2. **Refresh the page**: Some stores use custom domains that take a moment to detect
3. **Check for conflicts**: Other ecommerce spy extensions can interfere. Try disabling similar extensions temporarily
4. **Reinstall**: If all else fails, remove and reinstall the extension from the Chrome Web Store

### Login Issues

Some users report being logged out unexpectedly. This is typically caused by browser cookie clearing or using Chrome in incognito mode. To fix:

- Ensure the extension is allowed in incognito mode if you use it there (enable in `chrome://extensions/`)
- Log back in through the extension panel — your saved research syncs automatically

### Data Not Loading

If the extension panel opens but shows no data, the store may be recently launched (too new for traffic data to exist), or it may not be a Shopify store at all. Verify the store platform by checking for Shopify-specific elements in the page source.
